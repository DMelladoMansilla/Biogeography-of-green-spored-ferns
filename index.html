GLMM OVERALL Green-spores ferns

This code shows how to performe the GLMM for the overall richness proportion of chlorophyllous-spored fern species. Here I used glmmTMB since richness is a proportion (values 0 to 1) and therefore, follows a beta distribution. I also include the code to test for spatial autocorrelation. For the analyses of epiphytic and terrestrial proportions I followed the same workflow.

R2s table

Create a table with the r2 of each single-model

table_gs_prop2<- foreach(mod=sin_mod_prop, .combine = "rbind") %do% {
  m <- insight::get_variance(mod)
  r2_marginal <- m$var.fixed / (m$var.fixed + m$var.random + m$var.residual)
  r2_conditional <- (m$var.fixed + m$var.random) / (m$var.fixed + m$var.random + m$var.residual)
  r2_marginal$name <- as.character(mod[["call"]][["formula"]][[3]][[2]])
  return(r2_marginal)
}
